# Space-Shooter

* It's a simple arcade game for mac and andorid devices.
* Spaceship has to destroy as many asteroids as possible without colliding with them.

# Controls

* Android Devices - Accelorometer to steer the ship and touch to fire.
* Mac users - Arrow Keys to steer the ship and press 'F' to fire.
